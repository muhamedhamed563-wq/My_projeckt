# Enterprise Multi-Branch Network Simulation (CCNA)

## 🌐 Project Overview
A complete Cisco Packet Tracer simulation of a multi-branch enterprise network. This lab demonstrates advanced CCNA-level concepts including dynamic routing, security, and network services.

## 🛠️ Tech Stack
- **Routing:** OSPF (Area 0), Static/Default Routing.
- **Switching:** VLANs, Inter-VLAN Routing (ROAS), 802.1Q Trunking.
- **Security:** Extended ACLs, Port Security.
- **NAT:** PAT (NAT Overload) for Internet Egress.
- **Services:** DHCP Relay, DNS, HTTP, WPA2-PSK Wireless.

## 📂 File Structure
- `Multi-Branch Enterprise Network Infrastructur.pkt`: The original Cisco Packet Tracer file.
- `Multi-Branch Enterprise Network Infrastructur Document.pdf`: Full technical design and IP addressing plan.
- `Topology_Diagram.png`: High-level network map.

## 🚀 How to Run
1. Download [Cisco Packet Tracer](https://www.netacad.com).
2. Clone this repo or download the `.pkt` file.
3. Open the file and verify connectivity by pinging `8.8.8.8` from any branch PC.
